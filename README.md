# Sudoku Coach - Learn Section Screenshots

This repository contains screenshots of the **Learn** section from [sudoku.coach](https://sudoku.coach/en/learn), capturing the main content of each technique page for reference and offline use.

## Structure

Each subfolder corresponds to a language locale. Images are organized by language:
- **en/** — English
- **zh-cn/** — Chinese (Simplified)
- **de/** — German
- **fr/** — French

## Contents

Images of the learn pages, covering:
  - Rules and basics (candidate notation, difficulty, computer solver)
  - Basic techniques (naked single, hidden single, eliminate candidates, etc.)
  - Simple techniques (hidden pair, locked candidate, disjoint groups, etc.)
  - Advanced techniques (X-Wing, Swordfish, Skyscraper, Y-Wing, XYZ-Wing, etc.)
  - Hard techniques (Finned Swordfish, Jellyfish, BUG+1, X-Chain, etc.)
  - Brutal techniques (3D-Medusa, XY-Chain, AIC, Forcing Chains)
